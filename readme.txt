Audioboo WP is a plugin that allows you to display your latest boos from http://audioboo.fm

== Description ==

Audioboo is an iPhone and iPod Touch application that allows you to record audio, add photos and then saves it to the web complete with geo-location data. Audioboo WP allows you to display your most recent recordings from your account on http://audioboo.fm. You will need to create an account on Audioboo in order to be able to successfully link to it.

== Installation ==

1. Upload `audioboo_wp` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
